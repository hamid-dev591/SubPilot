# SubPilot

SubPilot is a modern, open-source subscription management platform designed to help users track, organize, and analyze their recurring expenses.

With SubPilot, users can manage subscriptions such as Netflix, Spotify, ChatGPT, Disney+, Prime Video, Microsoft 365, Adobe Creative Cloud, and any custom subscription from a single dashboard.

## Features

* Track recurring subscriptions
* Monthly and yearly spending analytics
* Upcoming payment reminders
* Category management
* Subscription status tracking
* Search and filtering tools
* Export and import data
* Local data storage
* Mobile-friendly design
* Fast and lightweight performance
* Open-source project

## Technology

* HTML5
* CSS3
* Vanilla JavaScript (ES6+)
* LocalStorage

## Why SubPilot?

Many people forget about active subscriptions and lose money on services they rarely use. SubPilot provides a simple and efficient way to understand recurring expenses and take control of subscription spending.

## Open Source

SubPilot is completely open source and welcomes contributions from developers around the world.
